General Statement

This statement relates to the privacy practices of this Web based application, henceforth referenced as the Application.

The Application respects your personal privacy rights and does not, as a general rule, collect personal information of any kind, without your clear permission. Any personal information which you volunteer will be treated with the highest standard of security and confidentiality, strictly in accordance with the principles of the General Data Protection Regulation (GDPR).

The Application contains links to other web sites, no responsability is taken for the privacy practices within any of these other web sites. It is recommended to read the privacy statements on other web sites you visit which may collect personally identifiable information.

This privacy statement applies solely to information collected on this Application.

Secure connection

The Application uses an encrypted connection over SSL (Secure Socket Layer) for your privacy and security.

Technical data

Some technical and necessary information such as IP address, userAgent, date and time are logged automatically for the purpose of maintaining the security and quality of the Application. These technical information are kept only for as long as is necessary to ensure the security and stability of the Application.

Cookie data

This Application uses essential and persistent cookies to support some of its core functions (i.e. Language, Cookie Consent).

You may Accept or Reject the usage of non-essential cookies (i.e. ShareThis). You can do so at anytime via the collapsable cookie banner or by selecting the appropriate settings on your browser. Non-essential cookies are available to improve your experience and to monitor the site traffic, however please note that if you reject the non-essential cookies then you may not be able to use the full functionality of this Application.

No Cookies are set to personally identify users anyway.

Analytics data

IP addresses are anonymised and information aggregated (combined) to show trends within the Application.

This data is retained to evaluate information on how the Application is consumed by users (humans), web-service requests (machines) and bots (search-engines).

It is not possible to identify users from analytics data.

Definitions
Browser

A browser is software that allows a user to access and view web sites / applications (i.e. Chrome, Firefox, Edge).

IP Address

IP Address stands for Internet Protocol Address. Each machine connected to the Internet has a unique address which can be dynamically or statically assigned. The IP address consists of four octects separated by dots (i.e 212.6.125.76).

Cookies

Cookies are small pieces of information, stored in simple text files in your internet browser by a web site or application. Session cookies expire when the browser is closed and persistent cookies have an expiry date.
