General Statement

This statement relates to the privacy practices of this Application.

The Application respects your personal privacy rights and does not, as a general rule, collect personal information of any kind, without your clear permission. Any personal information which you volunteer will be treated with the highest standard of security and confidentiality, strictly in accordance with the principles of the General Data Protection Regulation (GDPR).

The Application contains links to other web sites with no responsability for the privacy practices within any of these other web sites. It is recommended to read the privacy statements on other web sites you visit which may collect personally identifiable information.

This privacy statement applies solely to information collected on this Application.

Your personal data

The Application uses an encrypted connection over SSL (Secure Socket Layer).

Technical data

When you use this Application, as with most of the websites, some technical and usage information is logged automatically for analytical purposes.

Log data

Log data includes some of your details such as IP address, device, browser, date and time. This data is logged for the purpose of maintaining the security and quality of the Application and it is kept for as long as is necessary to ensure the security and stability of the Application.

Cookie data

This Application uses cookies to support some of its core functions (i.e. language preference, cookie consent). You may refuse the use of cookies by selecting the appropriate settings on your browser, however please note that if you do this you may not be able to use the full functionality of this Application.

No Cookies are set to personally identify users.

Analytics data

IP addresses are anonymised and information aggregated (combined) to show trends within the Application.

This data is retained to evaluate information on how the Application is consumed by users (humans), web-service requests (machines) and bots (search-engines).

It is not posisbile to identify users from analytics data.

Definitions

Browser

A browser is software that allows a user to access and view web sites / applications (i.e. Internet Explorer, Chrome, Firefox).

IP Address

IP Address stands for Internet Protocol Address. Each machine connected to the Internet has a unique address which can be dynamically or statically assigned. The IP address consists of four octects separated by dots (i.e 212.6.125.76).

Cookies

Cookies are small pieces of information, stored in simple text files in your internet browser by a web site or application. Session cookies expire when the browser is closed and persisten cookies have an expiry date.
